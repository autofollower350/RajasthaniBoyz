**Rajasthani Boyz Bot** ek fast aur lightweight WhatsApp MD bot hai, jo khaas karke **JNVU (Jodhpur)** ke students ki help ke liye banaya gaya hai. Iska main kaam Roll Number ya Form Number se turant **Admit Card** nikalna hai.

---

## ✨ Special Features

- **🚀 Instant Admit Card** – Bas Roll No. bhejo aur PDF pao.
- **🛠 Easy Setup** – No complicated config, simple pairing system.
- **🧹 Auto Cleanup** – RAM optimize karne ke liye purani files apne aap delete ho jati hain.
- **📱 Multi-Device Support** – Baileys v7 par chalta hai (Latest).
- **⚡ Super Fast** – Zero lag aur fast response system.
